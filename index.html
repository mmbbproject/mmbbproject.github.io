MMBB Template Blogger

Free! Dibangun menggunakan Bootstrap, front-end framewok paling hits sampai saat ini. Dengan dasar minimalist, responsive di banyak device dan mudah di custom menjadikan template ini layak untuk di coba.

Fitur

  • Disqus comment

    Pengaturan komentar jadi lebih mudah dengan Disqus, tampilan rapih, mendukung migrasi komentar jika mau berganti website/blog, atur urutan komentar dsb. Lihat fitur lengkapnya di Disqus Feature.

  • 3 Kolom

    Membagi halaman blog menjadi 3 baris kolom dengan ukuran berbeda dan jarak antar kolom lebih rapat. Juga posisi dan ukuran kolom bisa berbeda mengikuti lebar device (desktop, tablet, mobile), sangat responsive. Selengkapnya baca di jQueryHolic.

  • Breadcrumb 2 label

    Mendukung 2 label yang aktif di Breadcrumb, urutannya: label pertama > label kedua. Contoh: Home / Tech / Mobile / Judul postingan.

  • Sitemap

  • Related Post

  • Category widget

    Style widget Category/Label memakai element List Inline di padukan dengan element Button Bootstrap. Jika halaman label aktif, maka style label aktif di widget tersebut berubah menjadi button active.

  • Pager

    Style Pager memakai element Pager style next/prev, posisi pager melebar full. Dipadankan dengan icon MMBB-Font, setiap pager yang aktif atau tidak bisa di klik maka style berubah.

  • Status Message

    Style dan posisi Status Message lebih rapih, posisi berada di kolom utama yang sama dengan home post dan post detail sehingga jika ada Error search not found ataupun Error page not found maka kolom sidebar1 dan sidebar2 tetap pada tempatnya.

  • Font Icon

    Menghilangkan default font-icon Glyphicons, diganti dengan MMBB-Font icon, custom beberapa icon (kurang dari 50) menjadikan font-icon ini sangat ringan. Icon yang ada di MMBB-Font diantaranya:

Keunggulan

  • Mobile First

    Karena dibangun menggunakan Bootstrap, maka semua fitur Bootstrap pasti bisa di terapkan di MMBB Template ini. Bootstrap v3.3.7 sudah mendukung fitur mobile first, artinya tampilan dan performa lebih di utamakan untuk mobile. Istilah lainnya yaitu Mobile Friendly, untuk test kunjungi Mobile-Friendly Test Google.

  • Responsive

    Tampilan menyesuaikan dengan device yang di pakai (mobile, tablet, desktop). Posisi dan ukuran kolom (sidebar1, main page, sidebar2) bisa berubah dan berbeda di setiap device namun tetap rapih -tidak mendukung custom browser seperti UC browser dan Opera mini.

    Untuk test, buka halaman MMBB Demo dan masuk ke menu Developer di browser, caranya: Chrome tekan tombol keyboard Ctrl + Shift + I, untuk Firefox tekan Ctrl + Shift + M.

  • Full Custom

    Menghilangkan CSS dan Javascript bawaan Blogger, menghilangkan font-icon Glyphicon Bootstrap ( menggantinya dengan MMBB-Font, koleksi < 50 icons) dan menghilangkan komponen CSS (well, jumbotron) dari Bootstrap menjadikan MMBB Template ini sangat ringan.
